Add Missing Contributions

If you completed a Gitcoin Grants checkout, but don't see evidence of this in your email or the Gitcoin interface, you can use this form to fix that! This form supports both L1 transactions and L2 transactions (on zkSync).

{% csrf_token %}
Instructions

  • Make sure your wallet is connected to mainnet with the same address you used to checkout
  • If you donated using L1 (Standard Checkout), please enter the transaction hash. There is no need to also enter your wallet for L1 transactions. *Do not enter a transaction hash for L2 transactions*
  • If you donated using L2 (zkSync Checkout), please enter your wallet address, *Do not enter a transaction hash for L2 transactions*
  • At least one of these two is required
